# AbilitysPMMP-DevsES
**Español**: Un plugin de "abilitys" especiales para pvp. 
<br>
**English**: A plugin of "abilties" specials for pvp.

<br> 

## Todos los comandos / uso.


| **Comando-subcomando** | **Descripcion** | **Permiso** <br> nombre/acceso |
| :--- |:---| :---: |
|`/ability help`| <h6>Lista los comandos escenciales para el plugin. | `'abilityspmmp_cmd_abiltiy'` <br> op | 
|`/ability list`| <h6>Lista nombre y id de todas las abiliys. | `'abilityspmmp_cmd_abiltiy'` <br> op |
|`/ability debug <'on'/'off'>`| <h6>Activa el modo debug. <br> | `'abilityspmmp_cmd_abiltiy'` <br> op | 
|`/giveability <ability_id/'all'> <amount>`| <h6>Agrega la abilidad determinada en <ability_id> <br><br> *Nota:* <abilty_id> va entre '', ademas 'all' selecciona todas las abilities.</h6> | `'abilityspmmp_cmd_giveability'` <br> op |

## Otros.

- `Modo Debug`: El modo debug sirve para test abiltiys de tipo Throwing, al activar el "modo debug" las abilitys tipo throwing se comportaran como boosting, esto para probar su funcionamiento. Esto solo aplicara para los usuarios con op y las abilitys en este modo no tendran cooldown ni se restaran.
